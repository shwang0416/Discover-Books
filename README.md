# Discover-Books
Next.js 13^버전 프로젝트로 server action, tanstack-query(v5)를 사용하는 기술 실습 프로젝트로,
[itbook.store](https://itbook.store)의 책을 검색하고 상세 정보를 확인할 수 있는 도서 검색 페이지입니다.


## 주요 기능
- 도서 리스트 페이지, 클릭시 상세 정보 페이지
- 도서 리스트 무한 스크롤
- 리스트 검색 시 or(|) & not(-) 연산 포함
